# Laptop Performance Troubleshooting Guide

## Overview

Laptop performance issues may be caused by high CPU usage, memory pressure, low disk space, pending operating system updates, failing battery, or background applications.

## Common Symptoms

Users may report that the laptop is slow, freezes during meetings, applications take a long time to open, video calls lag, the fan is loud, or the battery drains quickly.

## Diagnostic Checks

The agent should check CPU usage, memory usage, free disk space, battery health, pending OS updates, and device age.

## Recommended Steps

If CPU usage is high, close unused applications, restart the laptop, and check for background update processes.

If disk space is low, remove temporary files, empty the downloads folder, and move large files to approved cloud storage.

If OS updates are pending, install updates outside working hours and restart the laptop after updates.

If battery health is poor, create a support ticket for hardware review.

## Escalation Rules

Create a support ticket if battery health is poor, disk free space is below 5 GB, CPU usage is above 90 percent, the issue disrupts meetings or business-critical work, or the device has repeated failures.

## Agent Instruction

The agent should provide user-level steps and create a ticket if device health indicates IT intervention is needed.
